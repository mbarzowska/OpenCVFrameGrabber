# OpenCVFrameGrabber

Używane skróty: \
OT - obecny tryb

Główne tryby pracy: \
C - kamera, przechwytywanie obrazu z kamery \
V - video, przechwytywanie obrazu z pliku video \
I - image, przechwytywanie obrazu z pliku graficznego 

Tryb PI - path input, wprowadzanie ścieżki
